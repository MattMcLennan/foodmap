VIEW
------------

As a user I should be able to see restaurant ratings

As a user I should be able to select a restaurant to view more details

As a user I should be able to read reviews

As a user I want to be able to to filter by meal (dinner, lunch, breakfast)

As a user I want to be able to to filter by city

As a user I want to be able to to filter by neighbourhood
